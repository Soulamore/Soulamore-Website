# Security Policy

## Supported Versions

Soulamore is a continuously deployed web application. We prioritize the security of our live production environment.

| Version | Supported |
| ------- | ------------------ |
| Live Site (main) | :white_check_mark: |
| Old Versions | :x: |

## Reporting a Vulnerability

We take the security of our community seriously. If you discover a security vulnerability, we appreciate your help in disclosing it to us responsibly.

Please report any security vulnerabilities via email to **contact.soulamore@gmail.com**.

- Please do not open a public issue on GitHub for security vulnerabilities.
- We will acknowledge your report within 48 hours.
- We will provide an estimated timeframe for addressing the vulnerability.
